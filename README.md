## Features
* **Multi-Page Navigation**: Seamless experience using React Router.

* **Global State Management**: Centralized student data management using Context API.

* **Reusable Logic**: Optimized code through custom hooks (useFetch, useForm).

* **Backend Integration**: Full CRUD operations supported by a local JSON Server.

* **Responsive Design**: Modular CSS styling for all device sizes.

* **Persistence**: Real-time data synchronization with local REST API.

* **Testing**: Reliable unit testing with Vitest.


## Tech Stack 
* React, TypeScript, Vite, React-Toastify.

## Quick Start 
1. `npm install`
2. `npm run dev`  # Run App
3. `npm run test` # Run Tests
